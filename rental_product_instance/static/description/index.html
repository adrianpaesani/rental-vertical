Rental Product Instance

This file has been generated on 2020-03-15-10-11-30. Changes to it will be overwritten.

Summary

Add product instances identified by serial number as unique rented objects

Description

Model product is extended as an instance that can be traced by serial number. An instance can only have one serial number, so that it is unique in the system. It can be used in rental orders, repair orders and maintenance orders.
